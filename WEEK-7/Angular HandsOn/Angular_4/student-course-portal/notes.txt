angular.json
Contains the Angular project configuration such as build, serve, assets, styles and output folder.

package.json
Contains project information, dependencies and npm scripts.

tsconfig.json
Main TypeScript compiler configuration.

tsconfig.app.json
TypeScript configuration used only for this application.

main.ts
Entry point of the Angular application.

app.config.ts
Contains global providers and configuration for the Angular application.

app.component.ts
Root component of the Angular application.

index.html
Main HTML page loaded by the browser.

Compiled Angular code is present inside main-NLVHSGWP.js

maximumWarning
Shows a warning if the application size exceeds the specified limit.

maximumError
Fails the build if the application size exceeds the specified limit.
